# 🎮 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.  
Play against the computer and track your score in real-time!

---

## 🚀 Features

- 🧠 Computer generates random moves
- 🎯 Real-time result display (Win / Lose / Draw)
- 📊 Live score tracking (User vs Computer)
- 🎨 Clean and responsive UI
- ⚡ Instant feedback with color changes
- 🖱️ Interactive click-based gameplay

---

## 🛠️ Technologies Used

- **HTML** – Structure of the game  
- **CSS** – Styling and layout  
- **JavaScript** – Game logic and interactivity

## 🎮 How to Play

1. Click on any option:
   - ✊ Rock  
   - ✋ Paper  
   - ✌️ Scissor  
2. Computer will randomly select its move  
3. Result will be displayed instantly  
4. Scores will update automatically
